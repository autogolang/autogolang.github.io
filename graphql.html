AutoGolang

Convert GraphQL schema to go struct

For Json conversion visit Json AutoGolang

This tool instantly converts GraphQL Schema into a Go type definition. Paste a GraphQL Schema structure on the left and the equivalent Go type will be generated to the right, which you can paste into your program. The script has to make some assumptions, so double-check the output!
