AutoGolang

Convert HTTP Request schema to go struct

For Json conversion visit Json AutoGolang

This tool instantly converts HTTP Request Schema into a Go type definition. Paste a HTTP Request Schema structure on the left and the equivalent Go type will be generated to the right, which you can paste into your program. The script has to make some assumptions, so double-check the output!
